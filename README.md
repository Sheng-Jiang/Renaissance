# Renaissance Developer Academy Courseware

Welcome to the courseware repository for the Renaissance Developer Academy.

This repository houses the lesson plans, slide decks, exercise guides, and project templates for the program. The curriculum is designed to transform mid-level engineers into forward-deployed polymath builders who own the outcome, communicate with precision, and ship high-quality, AI-augmented software rapidly.

## Course Structure

*   **/Background**: Core concept documents detailing the "Renaissance Developer" archetype.
*   **/module-1**: The full curriculum for Module 1, covering everything from the fundamental mindsets to hands-on mastery of our AI toolset (Claude Code, Kiro, Antigravity, Nanobot).

### Module 1 Outline

Module 1 is organized strictly by day to ensure a smooth, progressive learning experience:

1.  **Day 1: Concept & Vision.** Defining the archetype, the five mindsets, the AI multiplier model, and assessing our baseline.
2.  **Day 2: AI Tool Mastery.** Hands-on labs with Claude Code CLI, Kiro, and Antigravity, concluding with a comparative "Rosetta Stone" build.
3.  **Day 3: Agent Architecture.** Deep dive into autonomous agents via the OpenClaw ecosystem. Creating custom skills for the Nanobot agent and implementing CI/CD testing.
4.  **Day 4: The One-Day Ship Challenge.** A full-day sprint where students architect, build, test, and deploy a complete AI-augmented application to production in hours, ending with lightning demos.
5.  **Day 5: Reflection.** Constructing study notebooks in NotebookLM, revisiting the Day 1 baseline, and taking time for self-directed exploration ("Impossible Stuff Day").
6.  **Add-On: GitHub Bootcamp.** A foundational prep course covering Git, GitHub Actions, and Projects for students who need a workflow refresher before starting.

## Formats
- Slides present in standard Markdown (`.md`) format using Marp, complete with corresponding `.pptx` (PowerPoint) exports.
- All pre-reads and exercise guides are in clear, readable Markdown.
