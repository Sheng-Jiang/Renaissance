# Exercise: Extend Nanobot with a Custom Skill
**Time: 75 Minutes**

## Overview
In this exercise, you will modify the Nanobot architecture by writing a custom skill and proving it works.

## Task
Add a custom skill to your Nanobot agent.

### Options
1.  **A "Project Status" skill** that reads your GitHub repository's open issues and summarizes them for you via chat.
2.  **A "Study Buddy" skill** that quizzes you on technical concepts from a Markdown file you provide to it.
3.  **Custom:** Design your own skill based on a real problem you've encountered.

### Instructions
1.  Use **Claude Code CLI** to help write the skill.
2.  *Verification requirement:* Practice the AI Multiplier model. Do not blindly accept Claude's logic. Review every line. Does it handle API limits? Authentication? Bad user input?
3.  Test the skill via your configured messaging platform (Discord or Telegram).
4.  Push your modified Nanobot fork to GitHub. Create a Pull Request describing exactly what you built and why.
