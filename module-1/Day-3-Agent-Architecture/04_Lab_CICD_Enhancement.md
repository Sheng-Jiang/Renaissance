# Lab: CI/CD Enhancement for Agents
**Time: 60 Minutes**

## Overview
Autonomous agents must be governed strictly. Verification relies heavily on automated testing. You will enhance your GitHub Actions pipeline for your Nanobot fork today.

## Instructions
Enhance your GitHub Actions pipeline for AI-augmented development.

### Core Requirements
1.  Add automated testing for your new Nanobot custom skill.
2.  Add a linting step that checks the code quality of the AI-generated code.

### Advanced (Optional)
1.  Add a matrix build testing your skill across Python 3.10, 3.11, and 3.12.
2.  Add dependency caching for faster pipeline runs.

*Note for GitHub Add-On participants:* This is practice. *For those who skipped the Add-On:* This is your first real pipeline. 

### Peer Debugging
Troubleshoot with a partner. Debug each other's YAML workflows *before* asking an instructor for help.
