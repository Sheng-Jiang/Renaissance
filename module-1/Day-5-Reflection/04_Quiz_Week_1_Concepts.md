# Quiz: Week 1 Concepts

## Questions

1.  **Which of the following is NOT one of the five core mindsets of a Renaissance Developer?**
    A) Curiosity
    B) Specialization
    C) Precision
    D) Systems

2.  **In the AI Multiplier Model (Generation $\rightarrow$ Verification $\rightarrow$ Business Value), what role does the human engineer primarily play?**
    A) The Generation Engine
    B) The Business Value Consumer
    C) The Quality Gate
    D) The Deployment Manager

3.  **To orchestrate multiple AI agents in parallel on a complex task, which tool in our curriculum is best suited?**
    A) Kiro
    B) Claude Code CLI
    C) NotebookLM
    D) Antigravity

4.  **What defines an autonomous *agent* (like Nanobot) compared to a standard coding assistant?**
    A) It generates code faster.
    B) It maintains state/memory and can take independent action via external tool integrations.
    C) It relies purely on the IDE context window.
    D) It only uses a single LLM provider.

5.  **What is a critical threat vector identified in the OpenClaw ecosystem case study?**
    A) High latency responses.
    B) Prompt injection through malicious third-party skills.
    C) Inability to format JSON correctly.
    D) Code syntax highlighting errors.

6.  **In Kiro's Spec-Driven Development workflow, what is the sequence of artifact generation?**
    A) Tasks $\rightarrow$ Design $\rightarrow$ Requirements $\rightarrow$ Code
    B) Requirements $\rightarrow$ Design $\rightarrow$ Tasks $\rightarrow$ Code
    C) Design $\rightarrow$ Tasks $\rightarrow$ Requirements $\rightarrow$ Code
    D) Code $\rightarrow$ Requirements $\rightarrow$ Design $\rightarrow$ Tasks

## Answer Key

1.  **B) Specialization**. The mindset is *Polymath*, which champions cross-disciplinary learning, not narrow specialization.
2.  **C) The Quality Gate**. The human acts as the architect and the verifying quality gate.
3.  **D) Antigravity**. Antigravity's Manager View is designed for parallel multi-agent orchestration.
4.  **B) It maintains state/memory and can take independent action via external tool integrations.**
5.  **B) Prompt injection through malicious third-party skills.** This is a major security risk for autonomous agents executing untrusted tool logic.
6.  **B) Requirements $\rightarrow$ Design $\rightarrow$ Tasks $\rightarrow$ Code.**
