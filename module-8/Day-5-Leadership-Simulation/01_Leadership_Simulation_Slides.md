# The Leadership Simulation

You are the Technical Lead. A critical decision must be made today. **Three stakeholders disagree.**

Your job:
1.  **Listen** to each party genuinely.
2.  **Ask** the right clarifying questions.
3.  **Make** a recommendation grounded in evidence.
4.  **Defend** it under challenge — while acknowledging trade-offs.

The mark of technical leadership: every stakeholder feels **heard**, even if they don't get everything they wanted.

---

# Scenario Example

> A security vulnerability has been found in production.
> - **CISO:** "Immediate shutdown. No exceptions."
> - **Sales:** "We have a demo with a key prospect tomorrow."
> - **Engineering:** "We can patch it in 48 hours, but we'd need to skip code review."

There is no obviously right answer. The learning is in the **navigation.**
